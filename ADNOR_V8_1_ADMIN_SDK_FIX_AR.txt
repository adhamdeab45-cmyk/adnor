الحل:
1. أضيف دعم Firebase Admin SDK في server.js.
2. أضيفت dependency: firebase-admin.
3. /api/health صار يرجع ok حتى لو Firebase محمي، مع إظهار حالة firebase.
4. heartbeat عند تشغيل السيرفر صار لا يوقف السيرفر إذا Firebase رفض.
