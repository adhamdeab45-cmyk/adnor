ADNOR COMPLETE V6 FINAL
=======================

مهم جداً:
- لم يتم لمس نظام Gmail / Google login الذي أصبح يعمل عندك.
- لا تغيّر Firebase Auth ولا Google Cloud ولا Render Auth settings إذا كانت شغالة.

الموجود في هذه النسخة حسب الاتفاق:

1) تسجيل الدخول
- تسجيل Google/Gmail محفوظ كما هو.
- إخفاء واجهة كلمة السر العادية للمستخدم.
- تأكيد العمر والشروط قبل الدخول.

2) واجهة المستخدم
- إخفاء الإيميل العلني من الهيدر واستبداله باسم/ID مختصر.
- إزالة شارات النسخة/Server من واجهة المستخدم قدر الإمكان.
- قائمة سفلية مرتبة: اليانصيب، الألعاب، المحفظة، الدعم.
- معلومات الحساب الشخصية في الإعدادات وليس داخل المحفظة.

3) السحوبات
- شراء التذاكر في اليومي فقط.
- الأسبوعي/الشهري/السنوي للعرض والسحب والترحيل.
- أدمن: حفظ رقم فائز، تنفيذ الآن، وجدولة التنفيذ الساعة 8 مساءً.
- سيرفر 24H + /api/health.

4) المحفظة والإيداع والسحب
- إيداع مع رفع صورة إيصال.
- الرصيد لا يضاف إلا بعد موافقة الأدمن.
- أدمن: عرض الإيصال، قبول، رفض.
- طلبات السحب تظهر للأدمن.

5) الدعم
- المستخدم يرسل تذكرة دعم.
- التذاكر تظهر للأدمن.
- الأدمن يرد ويغير الحالة.

6) صندوق الحظ اليومي
- مرة كل 24 ساعة.
- عداد للمستخدم.
- أدمن يتحكم بالتفعيل، المبلغ الثابت، أقل/أعلى ربح، الاحتمال، ومكان الإضافة رصيد/بونص.

7) عجلة الحظ الملكية
- تصميم أفخم.
- قيم العجلة المطلوبة تقريباً: x1, x2, x4, x8, x16, x32, x64, x128, x236, x472, 5 تذاكر, x1000, x0.
- أدمن يتحكم بالخانات والاحتمالات.
- أدمن يخصص نتيجة لمستخدم عبر UID.

8) الدعوات
- كود دعوة لكل مستخدم.
- رابط دعوة.
- أدمن يتحكم بمكافأة صاحب الدعوة ومكافأة المستخدم الجديد.
- أدمن يضيف كود مخصص ويربطه ب UID ويحدد حد استخدام.

9) شاشة افتتاحية بعد الدخول
- أدمن يتحكم بصورة/رسالة/مدة ظهور 3 ثواني تقريباً.

10) حقن الأكواد
- إضافة أكواد بدون حذف القديم.
- تفعيل/إيقاف/أرشفة/استرجاع قدر الإمكان.
- حماية من كود JS يخرب الصفحة قدر الإمكان.

11) إعدادات Render
Build Command:
npm install --registry=https://registry.npmjs.org/

Start Command:
npm start

Health Check Path:
/api/health

Environment Variables:
FIREBASE_DATABASE_URL=https://adnor-vp-default-rtdb.firebaseio.com
DRAW_TIMEZONE=Europe/Istanbul
ADMIN_TOKEN=ADNOR_ADMIN_2026_987654
SESSION_SECRET=adnor_secret_2026
